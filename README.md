# Biorhythm Chart

Interactive biorhythm chart that visualizes your physical, emotional, and intellectual cycles based on birthdate. Renders animated sine-wave curves on an HTML5 canvas with critical day detection, tooltip hover details, and month-by-month navigation.

**[Live Demo](https://zebiv.com/biorhythm/)**

## Features

- Physical (23-day), Emotional (28-day), and Intellectual (33-day) cycle visualization
- Animated chart drawing with glow effects
- Critical day detection when cycles cross zero or converge
- Hover tooltip showing exact values for any day
- Today marker line
- Month navigation (previous, current, next)
- Current state cards with percentage and description
- Birthdate and name persistence via localStorage

## Tech

- Vanilla HTML, CSS, JavaScript (no build step, no dependencies)
- HTML5 Canvas 2D rendering with high-DPI support
- IBM Plex Mono typeface
- Dark theme with accent colors (#00ff88, #ff00ff, #00ccff)
